## 一、总体评分规则
总分 = 春季训练 $\times$ 10% + 暑期训练 $\times$ 60% + 网络赛 $\times$ 30%

## 二、春季训练评分规则
### 2.1 总体规则
- 比赛数据来自 HDU 平台，共 10 场个人训练赛
- 由于春季没有集中时间进行组队训练，因此组队训练不计入成绩
- 考虑到存在新生军训的情况，取所有场次中成绩最好的 70% 场次计入总成绩。新生赛前就已经参与训练的队员，取其最好 7 场的平均成绩；新生赛后参与训练的队员，取其最好 5 场的平均成绩。
- 成绩保留到小数点后 2 位

### 2.2 单场得分

```
得分 = 过题数 / baseline × (801 − 排名) / 800 × 100
```

- 排名使用 HDU 平台全场排名
- 若得分 < 0 或未参赛，按 0 分计算
- Baseline 题数：全场最高过题数，指所有属于 University 组中除特邀嘉宾外的最高过题数
- 排名基数为 800，理论上平均每场位于集训队内最后 10% 左右的选手，得分 = 0 分
- 存在疑似违规现象的选手，其当场成绩作废，得分按 0 分计算。如有需要，可在赛季训练群内申诉。

### 2.3 团队总成绩
- 队伍总成绩 = 队伍所有成员个人总成绩的平均值

## 三、暑期训练评分规则
### 3.1 总体规则
- 比赛数据来自 牛客、杭电 平台，各 10 场训练赛，总计 20 场。
- 取所有场次中成绩最好的 80% 场次计入总成绩，即取最好 16 场的平均成绩。
- 成绩保留到小数点后 2 位

### 3.2 单场得分
    
```
牛客得分 = 过题数 / baseline × (601 − 排名) / 600 × 100
杭电得分 = 过题数 / baseline × (501 − 排名) / 500 × 100
```

- 排名使用全场排名
- 若得分 < 0 或未参赛，按 0 分计算
- Baseline 题数：一般情况下指全场第 20 名队伍过题数，可能根据实际情况灵活调整。不取第1名队伍的过题数原因在于往年存在个别极强队伍（甚至有些不属于 Asia EC 赛区）过题数明显领先，导致大家的成绩被过度压缩，区分度不明显。
- 排名基数为牛客 600，杭电500，因为牛客参与队伍数较多。理论上平均每场位于集训队内最后 20% 左右的队伍，得分 = 0 分
- 存在疑似违规现象的队伍，其当场成绩作废，得分按 0 分计算。如有需要，可在赛季训练群内申诉。

### 3.3 团队总成绩
- 队伍总成绩 = 成绩最好的 80% 场次成绩的平均值

| 场次 | baseline |
|------|--------|
| 牛客1 | 9 |
| 杭电1 | 7 |
