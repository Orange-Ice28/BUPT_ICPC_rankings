## 一、总体评分规则
总分 = 春季训练 $\times$ 10% + 暑期训练 $\times$ 60% + 网络赛 $\times$ 30%

## 二、春季训练评分规则
### 2.1 总体规则
- 比赛数据来自 HDU 平台，共 10 场个人训练赛
- 由于春季没有集中时间进行组队训练，因此组队训练不计入成绩
- 考虑到存在新生军训的情况，取所有场次中成绩最好的 70% 场次计入总成绩。新生赛前就已经参与训练的队员，取其最好 7 场的平均成绩；新生赛后参与训练的队员，取其最好 5 场的平均成绩。
- 成绩保留到小数点后 2 位

### 2.2 单场得分

```
得分 = 过题数 / baseline × (801 − 排名) / 800 × 100
```

- 排名使用 HDU 平台全场排名
- 若得分 < 0 或未参赛，按 0 分计算
- Baseline 题数：全场最高过题数，指所有属于 University 组中除特邀嘉宾外的最高过题数
- 排名基数为 800，理论上平均每场位于集训队内最后 10% 左右的选手，得分 = 0 分
- 存在疑似违规现象的选手，其当场成绩作废，得分按 0 分计算。如有需要，可在赛季训练群内申诉。

### 2.3 团队总成绩
- 队伍总成绩 = 队伍所有成员个人总成绩的平均值
