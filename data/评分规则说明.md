# 评分规则说明

## 一、数据来源

- 比赛数据来自 HDU 平台，共 10 场训练赛（cid=1197~1206），搜索关键词"北京邮电"
- 组队信息来自 `team.xlsx`
- 每场比赛的 baseline（全场最高过题数）来自 `baseline.xlsx`
- team1790（胡恒瑞）数据已剔除，不参与排名

---

## 二、外榜（score_result.xlsx）

### 2.1 单场得分

```
得分 = 过题数 / baseline × (801 − 排名) / 800 × 100
```

- 排名使用 HDU 平台全场排名
- 若得分 < 0 或得分 > 100，按 0 分计算
- 排名为 0（未参赛）时，得分 = 0

### 2.2 特殊处理

- 周弋然（team0982）第 8 场成绩强制按 0 分计算

---

## 三、内榜（score_result_inner.xlsx）

### 3.1 单场得分

```
得分 = 过题数 / baseline × (总人数 − 内排名 + 1) / 总人数 × 100
```

- **总人数**：组队表中该场过题数 > 0 的人数
- **内排名**：在总人数中按过题数降序排列的名次（同分同名次）
- 若得分 < 0 或得分 > 100，按 0 分计算
- 未参赛或过题数为 0 时，得分 = 0

## 五、个人总成绩计算

- **队伍编号 ≤ team1791**：取所有比赛中最高的 **7 场**得分，计算平均分
- **队伍编号 > team1791**：取所有比赛中最高的 **5 场**得分，计算平均分
- 若所有场次得分均为 0，则总成绩为 0

---

## 六、队伍总成绩计算

```
队伍总成绩 = 队伍所有成员个人总成绩的平均值
```

- 队伍按总成绩从高到低排名
- 所有成绩保留到小数点后 2 位

---

## 七、输出文件

### 外榜：`score_result.xlsx`
### 内榜：`score_result_inner.xlsx`

每个文件包含两个工作表：

### Sheet 1：个人每场成绩

| 列 | 内容 |
|----|------|
| A | 排名（按个人总成绩降序） |
| B | 队伍编号 |
| C | 姓名 |
| D | 个人总成绩 |
| E~G | 第1场：过题数、排名、得分 |
| H~J | 第2场：过题数、排名、得分 |
| ... | ... |
| 共34列 | 排名 + 队伍编号 + 姓名 + 个人总成绩 + 10场 × 3项 |

- 组队表中成员排在前半部分，非组队表成员在空行后，各自按个人总成绩降序排列
- **灰色底色**：未计入个人总成绩的场次（即不在最佳 N 场之列）

### Sheet 2：队伍总体情况

| 列 | 内容 |
|----|------|
| A | 排名 |
| B | 中文队名 |
| C | 英文队名 |
| D~E | 队员1姓名、总成绩 |
| F~G | 队员2姓名、总成绩 |
| H~I | 队员3姓名、总成绩 |
| J | 队伍总成绩 |
